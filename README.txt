README

Project Title: ENN - AI Summarisation & Credibility Console App
SID: 2345045
Team: Ecaps

Instructions:
- Run ENNApp.java
- The app simulates processing an article, provides 3 summarisation modes, and scans for credibility

System Requirements:
- Java 8 or above
- IDE or terminal with `javac` and `java` support

Files:
- ENNApp.java (Main application)
- Article.java (Represents the article data)
- Summarizer.java (Processes article to create summaries)
- CredibilityScanner.java (Evaluates trust level of article)
